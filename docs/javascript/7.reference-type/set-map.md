- **Set**: value 不重复，没有 `get` ，只有 `has`
- **Map**: key 可以为任何类型， `get` 是引用比较，但 0相等、NaN相等。
- **WeakSet**: value 只能是对象，GC 不考虑对象的引用
- **WeakMap**: key 只能是对象，GC 不考虑对象的引用
