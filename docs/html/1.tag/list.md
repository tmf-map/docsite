## 描述列表

描述列表是一个包含术语定义以及描述的列表，通常用于展示词汇表或者元数据 (键-值对列表)。

### `<dt>...</dt>`

### `<dd>...</dd>`

### `<dl>...</dl>`

```jsx live
<dl>
  <dt>Coffee</dt>
  <dd>Latte</dd>
  <dd>Mocha</dd>
  <dt>Tea</dt>
  <dd>Green Tea</dd>
  <dd>Black Tea</dd>
</dl>
```
